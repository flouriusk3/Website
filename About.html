About

The iOS Learning Club is an educational website created to assist students and potential developers with learning iOS programming. The home page familiarizes the user with Swift and Xcode, which are both needed to complete the applications.

The applications section features three simple, easy-to-follow applications. The first is a monetary conversion application that allows the user to specify how many pennies, nickels, dimes, and quarters they want to convert to dollars. The second project is a degree conversion application. It converts Fahrenheit temperatures to Celsius. Lastly, the third application is an elementary calculator that allows the user to add, subtract, multiply, and divide two numbers. The iOS Learning Club teaches the users how to complete the projects through instructional pictures and step-by-step directions. For each project, the website includes related videos, Github links, and other relevant resources for additional help.

Kiyunna Flourius, Aisha Ayaz, Bryan Campos, Bobby Maiga, and Brooke Sextion collaborated to develop the iOS Learning Club website. Campos was responsible for the HTML and CSS design of the website. Maiga carried out the JavaScript design and server scripting components. Sextion executed the database, hosted and deployed the website, and contributed to the iOS projects. Ayaz analyzed the website and devised future research opportunities. Flourius facilitated the project, contributed to the HTML and CSS design, and assisted with the iOS programming applications.
