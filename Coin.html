Coin Converter

This application is a coin converter. The user enters any number of quarters, dimes, nickels, and pennies. The application will convert the number of coins to a dollar amount.

Instructions

  1. Open Xcode and create a new project. Name the application CoinConverter and make sure you’re using the swift language.
  2. Go to main.storyboard. In the bottom right corner, go to the object library. Add five labels, four text fields, and one button onto the view.
  3. From top to bottom, rename the five labels as stated below.
    • Quarters:
    • Dimes:
    • Nickels:
    • Pennies:
    • Total: (resize if needed)
  4. Rename the button “Calculate”.
  5. Your main.storyboard should look like the picture above.
  6. Go to the ViewController.swift file to add your variables.
    • @IBOutlet weak var quarterText: UITextField!
    • @IBOutlet weak var dimesText: UITextField!
    • @IBOutlet weak var nickelsText: UITextField!
    • @IBOutlet weak var penniesText: UITextField!
    • @IBOutlet weak var totalLabel: UILabel!
  7. Make the connections from your text fields/label by going to the connection inspector. Then control-drag from the small circle next to referencing outlets to the text field. Designate the appropriate outlet for the text field. Repeat for each text field and total label.
  8. Add an action function. This function will take the user input and convert the amount of coins to dollars.

    @IBAction func Calculate(_sender: Any) {
    // Convert user input from string to double
    Let quarters = Double(quarterText.text!)
    Let dimes = Double(dimesText.text!)
    Let nickels = Double(nickelsText.text!)
    Let pennies = Double(penniesText.text!)
    // If all text fields have user input, calculate the dollar amount
    If quarters != nil && dimes != nil && nickels != nil && pennies != nil {
    Let total = Double(quarters! * 0.25 + dimes! * 0.10 + nickels! * 0.05 + pennies * 0.01)
    totalLabel.text = “Your total is: $\(total).”
    }
    }

  9. Make the connection from your button to the calculate function. Control-drag from the button to the view controller. Then, select Calculate. Now, when you press the calculate button, the function will execute.
  10. Build and run your application. Enter numbers for all text fields and press calculate. Your total should display in the total label. You’re all done.
